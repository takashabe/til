## ENUM

いわゆる列挙型。列挙された値のリストで有効なデータ値を指定可能になる。

#### 定義

カラムの定義時に指定する

```
CREATE TABLE Bugs (
    ...
    status ENUM('NEW',  'IN PROGRESS',  'FIXED')
);
```
